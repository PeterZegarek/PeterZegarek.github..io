Elemental Hero

This is hands down my biggest as well as favorite project that I have so far worked on. This was a project for the SER225 class at Quinnipiac. This was a semester long project where the class split up into teams of 4 developers as well as one manager (a student who already took the course and was currently enrolled in a separate project management course). This project was run with the Agile/Scrum Framework in mind. We had weekly stand up meetings, and we had a total of 5 sprints throughout the semester.

I'll let the website for the game speak for itself, but in a nutshell, it's a platformer where you unlock new abilities and progress through levels until you face the final boss. This was all programmed in Java.

Online Trivia Game

This was one of my more recent projects, being for my CSC340 Networking and Distributed Processing Class. This was also a group project, where I as well as two other developers worked on using the TCP and UDP protocols together to make a functional multiplayer trivia game. Our game's theme was Star Wars, and participants can hit a buzzer to get the opportunity to answer the question (that every player views at the same time), but only the first person to buzz gets to answer. Our game handles multiple use cases, including players joining / leaving mid game, buzzing constantly, cases where no one buzzes, cases where someone refuses to answer, etc.

This project really challenged us, but it turned out to be a great game and I finished it feeling extremely confident using the TCP protocol and much more confident using UDP than before.
